helloworld
==========

Hello World in a bunch of languages

### Erlang

	% erl -compile hello
	% erl -noshell -s hello hello_world -s init stop

### Go

    % go run hello.go

### Groovy

    % groovy hello.groovy

### Haskell

    % runghc hello.hs

### Java

    % javac Hello.java && java Hello

### Python

    % python hello.py

### Scala

    % scala hello.scala
